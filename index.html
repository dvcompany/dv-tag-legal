디브이 태그 매니저 개인정보처리방침

최종 갱신일: 2026년 4월 30일
적용 대상: Chrome 확장 "디브이 태그 매니저"
운영자 연락처: dvcompany.dev@gmail.com

본 확장은 사용자의 개인정보 보호를 최우선으로 합니다. 본 방침은 확장이 어떤 데이터를 수집·사용·저장하는지 투명하게 설명합니다.

1. 수집·처리하는 정보

1-1. 라이선스 인증을 위해 외부 서버로 전송되는 정보

확장의 정상 동작을 위해 라이선스 검증 시 다음 정보가 운영자가 운영하는 Supabase 서버로 전송됩니다.

항목 설명 사용 목적
라이선스 키 사용자가 입력한 UUID 형식 키 정상 라이선스 여부 확인
디바이스 ID 확장이 최초 실행 시 생성하는 임의 UUID (사용자 식별 정보 아님) 라이선스 당 허용 디바이스 수 관리
User-Agent 브라우저가 제공하는 기본 문자열 디바이스 등록 시 1회, 환경 통계
  • 위 정보 외에 이름·이메일·전화번호·결제 정보 등 식별 가능한 개인정보는 수집하지 않습니다.
  • 라이선스 검증 결과는 5분간 로컬에 캐시되어 불필요한 서버 호출을 줄입니다.

1-2. 사용자 브라우저에만 저장되는 정보

다음 정보는 chrome.storage.local(사용자 PC)에만 저장되며, 외부로 전송되지 않습니다.

  • 사용자가 설정한 마켓 프리셋·태그 선택 상태
  • 디바이스 ID, 라이선스 키, 검증 캐시
  • 사용자가 입력한 네이버 검색광고 API 자격증명 (Customer ID, Access License, Secret Key)
  • 검색·동의어·노출량 조회 결과 캐시

1-3. 사용자 본인의 권한으로 호출되는 외부 API

확장은 사용자가 네이버 쇼핑 검색 결과 페이지를 보는 동안 다음 API를 호출하지만, 응답 데이터는 사용자의 브라우저 안에서만 처리되며 운영자 서버로 전송되지 않습니다.

  • 네이버 쇼핑 검색 API (search.shopping.naver.com) — 사용자의 기존 세션·쿠키 사용
  • 네이버 검색광고 API (api.searchad.naver.com) — 사용자가 직접 입력한 본인 API 자격증명 사용

운영자는 이 API 호출의 내용·결과에 접근할 수 없습니다.

2. 정보의 보유 및 파기

  • 외부 서버 저장 정보: 라이선스가 유효한 동안 보유되며, 라이선스 해지 또는 사용자 요청 시 즉시 삭제합니다.
  • 로컬 저장 정보: 사용자가 확장을 제거하거나 Chrome의 사이트 데이터 삭제 기능을 사용하면 즉시 함께 삭제됩니다.

3. 제3자 제공·판매

  • 확장은 수집된 데이터를 제3자에게 판매·임대·제공하지 않습니다.
  • 수집 데이터는 라이선스 인증 외 목적(예: 광고, 신용 평가, 마케팅 분석)으로 사용되지 않습니다.
  • 데이터 처리 위탁: 라이선스 인증을 위한 데이터베이스 호스팅에 한해 Supabase Inc. 의 인프라를 사용합니다.

4. Chrome 확장 권한 사용 근거

권한 사용 이유
activeTab, scripting 사용자가 보고 있는 네이버 쇼핑 페이지에 결과 패널 삽입
storage 위 1-2 항목의 로컬 데이터 저장
host_permissions (네이버 도메인) 사용자 세션으로 네이버 API 호출
host_permissions (*.supabase.co) 라이선스 검증 RPC 호출
