네이버 쇼핑 SEO 분석 도구 (Chrome 확장)

네이버 쇼핑 검색 결과 페이지에서 상품 태그, 키워드 검색량, 동의어, 카테고리 분포를 한 화면에서 분석합니다. 셀러가 검색 노출에 어떤 키워드·태그를 등록해야 할지 빠르게 판단하도록 돕는 도구입니다.

주요 기능

  • 검색 결과 태그 풀 자동 구성 - 검색 API의 제조사 태그와 상품 상세 페이지의 셀러 등록 태그를 합쳐 검색어별 태그 풀을 제공
  • 키워드 검색량 표시 - 네이버 검색광고 API와 연동해 월간 검색량을 칩 형태로 표시
  • 사전 등록 여부 확인 - 검색에 실제로 반영되는 태그만 강조해 셀러가 등록할 가치가 있는 키워드를 식별
  • 동의어·카테고리 분석 - 같은 의미의 다른 표현이나 카테고리 미분류 태그를 그룹으로 묶어 노출
  • 상품 순위 분석 (셀러센터 브랜드 스토어 로그인 시) - 내 스토어의 카테고리별 순위 노출 빈도 확인

다운로드 / 사용

본 확장은 Chrome Web Store를 통한 정식 출시를 준비 중입니다. 출시 전까지는 승인된 사용자에게 직접 배포되며, 사용 문의는 아래 연락처로 부탁드립니다.
